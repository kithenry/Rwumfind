* Rwumfind

* Description
- A flask web app for sharing rental posting links and finding room mates by virtue of who's looking for the same room as someone else.

* Capabilities
- User can share an external link.
- System notifies user in case someone else posts exactly the same link

* Installation
- Clone this repository
- Cd into the downloaded folder
- Create a virtual environment
- Activate the virtual environment
- Install flask with pip
- Cd into src/
- Run `flask app.py`
- Visit the url that'll be displayed in your terminal
